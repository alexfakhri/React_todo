React TodoApp
=====
A basic todo list app built with React, NPM, Bable and webpack.

Technologies Used:
------
- React
- NPM
- Bable
- Webpack

How to use:
----------

Clone the repository:
```shell
$ git clone https://github.com/alexfakhri/React_todo.git
```

Change into the directory:
```shell
$ cd React_todo
```

Install dependancies:
```shell
$ npm install
```

Start the server:
```shell
$ open src/index.html
```
